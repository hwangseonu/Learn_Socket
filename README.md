Learn_Socket (Written by Go)
==
Socket io 공부를 위한 서버입니다.

사용법
---

>> 빌드된 파일 이용  
>> 자기의 환경에 맞는 파일을 다운로드 합니다.  
>> ./파일이름 으로 실행합니다. (윈도우의 경우 .\\)

```go get github.com/hwangseonu/Learn_Socket```  
```cd $GOPATH/src/github.com/hwangseonu/Learn_Socket```  
```go build```  
```./Learn_Socket```
